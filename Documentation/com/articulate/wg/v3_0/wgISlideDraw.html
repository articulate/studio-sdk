AbsoluteId:String  [read-only]

The absolute id of the slide draw. An absolute id is a string containing the ids of the slide draw's ancester containers. It is in the form: [GRANDPARENT CONTAINER ID].[PARENT CONTAINER ID].[SLIDE DRAW ID].
